## Quick Start

### Clone this repo

```bash
$ git clone https://github.com/epilande/p2pwnme-space-wars.git
$ cd p2pwnme-space-wars
```

### Install dependencies

```bash
$ yarn install
```

### Launch

**Start web socket server:**

```bash
$ yarn start
```

**Start dev env:**

```bash
$ yarn dev
```
